数组-array

数组的基本性质:

  • 数组本身不属于基本数据类型,需要导入array使用数组
  • 数组内部元素类型相同
  • 数组不同于列表:内部元素相同,列表内部元素可以不同
  • 人如其名,内部元素只能是数字

    数组定义

  • 声明一个数组,必须遵循基本语法:
    arrayName = array(typecode, [initializers]): typecode:str
  • 当声明数据类型与initializers数据类型不一,则报错:
  • 指定数据类型后,initializers可为空:

    访问数组

    索引–[]

    切片–:

    index(x)

    返回首次出现x的最小索引

    负索引

    修改数组

    append()

    将一个元素添加到数组末尾

    insert(x, i)

    将元素x指定添加到i处

    extend(x)

    将x元素添加到数组,若x时数组,则二者应有相同的数据类型

    fromlist()

    将list中的元素添加到数组中

    reverse()

    反转数组

    +

    *

    将数组复制添加到本身

    del arr[i]

    删除某个元素,或数组

    remove()

    从数组中删除第一个出现的x元素

    pop()

    删除并返回数组的最后一个元素 也可指定pop对象索引
