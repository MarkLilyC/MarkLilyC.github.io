b = [0 1 2 3 4]
a = b[x : y]: 两索引各自计算,正值从左侧开始,负值从右侧开始

  • 左侧索引元素在右侧索引元素左侧,则返回非空
  • 某侧无索引则代表,从另一侧索引顺方向计数完
  • a为两侧索引之内交集元素返回的array
  • 若两侧索引之内无元素,则a为空array

    a = b[4:2] : a = []
    a = b[-3:4] : a = []
    a = b[4:-2] : a = []

  1. x>0 y>0
  • a = b[x:y]: a = x-y内元素返回array
  • a = b[ :y]: a = 0-y内元素返回
  • a = b[x: ]: a = x-len(b)内元素返回
  • x < y : a = []
  1. x<0 or y<0
  • a = b[-4:-1]: a = b中-4号元素到-1号元素返回
  • a = b[-4:1]: a = []
  • a = b[-4:]: a = -4 - 0索引之间元素返回
  • a = b[:-4]: a = []
  • a = b[3:-4]: a = []
  1. a = b[::a]
  • 在b中没隔a取一个元素返回a
  • 取值方向由a正负决定
