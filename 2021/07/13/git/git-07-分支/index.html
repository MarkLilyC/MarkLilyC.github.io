git-07-分支基础

git中对分支的操作是进行代码管理的重要功能

当某一个版本的代码性能稳定时 ，我们需要新建一个基于此主分支 此版本的分支进行开发

在分支上进行开发的一些好处有 ：

  • 分支之间相互独立 ，分支上的开发不会影响到主分支上已经稳定的代码
  • 多个分支可以提供给多个开发者进行不同模块的开发
